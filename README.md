# 🛒 SupportHub Commerce API

> 🚀 A secure RESTful e-commerce API built with Spring Boot, JWT Authentication, PostgreSQL, and Swagger.

---

# 📖 About the Project

SupportHub Commerce API is a backend application developed to simulate a modern e-commerce platform. It provides secure authentication, product management, and order processing while following RESTful principles and clean architecture practices.

The main goal of this project is to demonstrate backend development skills using the Spring Boot ecosystem.

---

# ✨ Features

- 🔐 User registration and authentication
- 🎫 JWT Authentication
- 👥 Role-based authorization (USER / ADMIN)
- 📦 Product management (CRUD)
- 🛍️ Order management
- 🔄 Order status workflow
- ⚠️ Global exception handling
- ✅ Bean Validation
- 📄 Pagination
- 📚 Swagger / OpenAPI documentation
- 🧪 Unit tests with JUnit 5 and Mockito
